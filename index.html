About The Event

Join us for an unforgettable experience where you'll solve problems and create solutions using the latest AI technologies!

Competition Categories

Best use of MCP server with SmythOS

Build an MCP (Model Context Protocol) server using SmythOS's SRE (Smith Runtime Environment) and explore compelling use cases, such as controlling other apps—like GitHub or Slack with AI agents.

MCP Server SRE AI Agents

Best use of Agent LLM with SmythOS

Unleash the power of SmythOS in creating a mixture of experts, building agents that use multiple AI types, routing requests to different workflows, and publishing them as Agent LLMs for other projects.

Agent LLM Mixture of Experts Workflows

Best use of edge/IoT/Raspberry Pi/local with SmythOS

Explore the Smyth Runtime Environment (SRE) running locally on devices like Raspberry Pi or thermostats, and uncover creative applications for local agents.

Edge Computing IoT Local Agents
More than one hire possible

Best use of SaaS building with SmythOS

Discover how to build your own SaaS (Software as a Service) applications with SmythOS, leveraging its API, Agent LLM, and MCP server access, and contribute to https://smythos.com/forged.

SaaS API SmythOS Forged

Best use of SRE code agents with SmythOS

Creating AI agents with SmythOS's SRE (https://www.npmjs.com/package/@smythos/sre) using code, with examples like integration with Llama Index, and contribute innovative use cases.

SRE Code Agents Llama Index

Best contribution to SRE open source

Contribute to the SmythOS open-source project (https://github.com/SmythOS/sre), demonstrating skill, aptitude, and creative, out-of-the-box, ambitious ideas with the SmythOS toolkit (SaaS and code). All entries should be submitted to https://smythos.com/forged, participate in the Discord community, and leave a GitHub star on the repo.

Frequently Asked Questions

Who can participate in Hack The AI?

Hack The AI is open to university students from any institution across Bangladesh. Whether you're a beginner or an experienced developer, all skill levels are welcome to participate and learn.

How many members can be in a team?

Each team can have a maximum of 3 members. This encourages collaboration while keeping teams manageable and ensuring everyone can contribute effectively.

What technologies can we use?

You can use any programming language, framework, or AI/ML tools. Popular choices include Python, TensorFlow, PyTorch, OpenAI APIs, and various cloud platforms. The focus is on solving problems using AI technologies.

Is this a virtual or in-person event?

Hack The AI is a hybrid event. The preliminary and final rounds (online) will be conducted virtually, while the grand finale will be held in-person at Green University of Bangladesh.

What are the prizes?

Winners will receive cash prizes (1st: $300, 2nd: $200, 3rd: $100), remote job opportunities with $700+ monthly salaries, premium access to SmythOS tools worth $3M+, and other exciting rewards.

Do I need to have AI experience?

No prior AI experience is required! We welcome participants of all skill levels. The hackathon is designed to be a learning experience, and we'll provide resources and mentorship to help you succeed.

How do I register?

Registration is simple! Click the "Register Now" button in the navigation or hero section, which will take you to our Google Forms registration page. Make sure to register before the deadline on September 10th, 2025.

What should I bring?

For the virtual rounds, you'll need a computer with internet access. For the in-person finale, bring your laptop, charger, and enthusiasm! Food and refreshments will be provided during the event.
